      NOW LETS PLAY WITH CSS

      CSS stands for Cascading StyleSheet and this will deal with the design of the HTML document.

      There are different was of implementing CSS to your document.

      • INLINE CSS within the same line of the HTML element using style="proptery=value;"
      • INTERNAL CSS uses the STYLE tag at the head or the body of the same HTML document.EXAMPLE: style="color:red" changes color of text for tag
      • EXTERNAL CSS uses a separate resource to provide the modifications of the HTML elements.

      It is possible to group and give tag multiple classes.

      • For instance, you can have p, h1, div{background-color:purple;} and it will be valid.

      A CLASS will be given to a multiple tags that want to have the same style while ID will be given to a special unique element

      So what are the types of styles for CSS?
      • COMMENTS: /**comment**/
      • BACKGROUND-COLOR: will color the background of the element

        • ex: I like to eat chicken wings.

      • COLOR: will color the text of the element

        • ex: I like to eat chicken wings.

      • TEXT-ALIGN: will align the text of the element

        • ex: I like to eat chicken wings.

      • FONT-FAMILY: will change the font of the element

        • ex: I like to eat chicken wings.

      • FONT-SIZE: will change the font of the element

        • ex: I like to eat chicken wings.

      • FLOAT: takes away the blocking style of elements and lets them float so the other elements move around it

        • ex: I like to eat chicken wings.

      • WIDTH: width of an image
      • HEIGHT: height of an image
      • MARGIN: space outside of the border
      • PADDING: space between border and element

        • ex: I like to eat chicken wings.

      • MAX-WIDTH: improves handling with small windows for text

        • ex: I like to eat chicken wings.

      • OVERFLOW: makes sure everything fits in the threshold of the screen

      • EVENT LISTENER: attaching an event to an element when something happens to the element, it will change.
      • POSITION: the position of the elements
        • FIXED: element is positioned relative to browser screen

          • (FIXED POSITION) ex: I like to eat chicken wings.

        • RELATIVE: positioned relative to normal position

          • (RELATIVE POSITION) ex: I like to eat chicken wings.

        • ABSOULTE: positioned relative to first ancestor

          • (ABSOLUTE POSITION) ex: I like to eat chicken wings.

        • STATIC: positioned according to flow of page

          • (STATIC POSITION) ex: I like to eat chicken wings.

      • DISPLAY: specifies how an element is displayed
        • INLINE: default value that displays element as inline element across the screen taking up as much space as they need

          • ex: I like to eat chicken wings.

        • BLOCK: displays element as block element

          • ex: I like to eat chicken wings.

        • INLINE-BLOCK: displays as inline level blocks

          • ex: I like to eat chicken wings.

      • BACKGROUND-IMAGE: allows the background to be an image like the design currently displayed.
